Este foi um projeto acadêmico feito durante uma aula de programação no Instituto Federal Catarinense (IFC) Campus São Bento do Sul com o objetivo de testar as habilidades de utilizar a GUI e fazer uma integração com um banco de dados pequeno.

O projeto consistia basicamente na criação de um banco de dados com duas ou três tabelas de um tema escolhido pelo aluno e criar um sistema em java que fizesse as operações básicas de inserção, atualização, vizualização e deleção dos dados.

O tema escolhido foi o do uma locadora de carros com tabelas para o carro, cliente e uma ligação n:n para eles.
